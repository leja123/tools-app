# Tools app

Web app with tools such as a calculator, a stopwatch, notes, to-do lists,...

## How to use:
### 1. Install xampp
Download it here: https://www.apachefriends.org/download.html (you need PHP and SQL)
### 2. Place it in the right directory
Clone the repo inside `C:\xampp\htdocs`
### 3. Create db
Create a new db called `tools_app`
### 4. Open localhost
In your browser go to `http://localhost/tools-app/`
